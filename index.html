Projetos de Robótica com Programação C++

Funcionalidades: Sensores de Linha: O robô utiliza sensores infravermelhos para detectar a linha e ajustar sua trajetória. Interface de Controle: Utilizei C++ para configurar a lógica de controle e integração com o hardware do robô. Interface de Usuário: Criei uma interface simples para programar e controlar o braço robótico, possibilitando a execução de tarefas automatizadas. Algoritmo de Cinemática Inversa: Implementado para calcular as articulações necessárias para posicionar a ponta do braço robótico em coordenadas específicas. Controle de Motores: Programação para controlar os motores do drone e gerenciar a resposta ao comando do piloto.
